### Examples of Detail Levels

**Minimal (AI fills in gaps):**
```
Topic: Introduction to Python Programming
Slides: 8
```

**Recommended:**
```
Topic: Python for Data Analysis: Pandas and NumPy for Beginners

Audience: Analysts transitioning from Excel

Slides: 12

Include: Setup instructions, data loading, cleaning techniques, basic visualizations, and real dataset examples
```

**Maximum Detail:**
```
Topic: Advanced Python for Financial Analysis: Building Algorithmic Trading Strategies

Audience: Quantitative analysts with basic Python knowledge

Slides: 16

Tone: Professional but approachable

Type: Technical

Cover: Market data APIs, pandas for time series, NumPy for calculations, backtesting frameworks, risk management, performance metrics, code examples for moving averages and RSI indicators, visualization with matplotlib, deployment considerations, regulatory compliance notes, and resources for continued learning
```
